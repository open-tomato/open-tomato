    Function formatMemoriesAsMarkdown

    • formatMemoriesAsMarkdown(memories: Memory[]): string

      Formats an array of Memory objects into a prompt-injectable markdown block.

      The output begins with a ## Agent Memories header, followed by each memory rendered as a blockquote. A Tags: annotation line is appended inside the blockquote when the memory has at least one tag. Memory blocks are separated by a blank line to allow truncateToBudget to cut at block boundaries.

      Parameters

      • memories: Memory[]

        The memories to format. May be empty.

      Returns string

      A markdown string ready for prompt injection, or an empty string when memories is empty.

      Example

      ## Agent Memories

      > Use Drizzle ORM for schema migrations.
      > Tags: orm, database

      > Always validate at system boundaries.
      > Tags: validation
